Detailed Description

This module provides an API over the lex finite state machine, generating tokens from multibyte character scripts.

◆ scallop_lang_token_next_raw()

struct scallop_lang_token scallop_lang_token_next_raw ( struct scallop_lang_token  previous)
inline

Returns the next, raw token in the script referred to by previous.

Raw tokens contain raw types, such as scallop_lang_lex_double_quote. Separate tokens are returned for beginning quote, the quoted word, and end quote, as well as any consecutive word tokens without separators.

It is recommended to use scallop_lang_token_next(), which will return a single scallop_lang_lex_word token in that scenario.

Parameters
previousA token returned by scallop_lang_token_init() or scallop_lang_token_next_raw().
Returns
A new token.
